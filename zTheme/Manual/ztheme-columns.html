Columns

Reorder and offset columns for flexible layouts

Column Ordering

Change visual order without changing HTML structure. The columns below are ordered 1, 2, 3 visually, but in the HTML they're 3, 1, 2:

First in HTML
.zOrder-3
Second in HTML
.zOrder-1
Third in HTML
.zOrder-2 
<div class="zRow">
  <div class="zCol zOrder-3">First in HTML</div>
  <div class="zCol zOrder-1">Second in HTML</div>
  <div class="zCol zOrder-2">Third in HTML</div>
</div>

Available Order Classes

Class Effect
.zOrder-first Move to beginning (order: -1)
.zOrder-0 to .zOrder-5 Set specific order (0-5)
.zOrder-last Move to end (order: 13)

Responsive Ordering

Change order at different breakpoints. Resize your browser to see the sidebar and main content swap positions:

Sidebar (First on mobile)
.zOrder-md-2
Main Content (Second on mobile)
.zOrder-md-1 
<div class="zRow">
  <div class="zCol-12 zCol-md-4 zOrder-md-2">Sidebar</div>
  <div class="zCol-12 zCol-md-8 zOrder-md-1">Main</div>
</div>

Available for all breakpoints: .zOrder-sm-*, .zOrder-md-*, .zOrder-lg-*, .zOrder-xl-*, .zOrder-xxl-*

Column Offsets

Push columns to the right by skipping column spaces. Think of offsets as invisible columns that create gaps:

.zCol-4 .zOffset-4
.zCol-3 .zOffset-3
.zCol-3 .zOffset-3 
<!-- Centered column -->
<div class="zRow">
  <div class="zCol-4 zOffset-4">Centered</div>
</div>

<!-- Two columns with gaps -->
<div class="zRow">
  <div class="zCol-3 zOffset-3">Column 1</div>
  <div class="zCol-3 zOffset-3">Column 2</div>
</div>

How Offsets Work

Available offsets: .zOffset-1 through .zOffset-11

Each number = columns to skip. Example: .zOffset-3 skips 3 columns (25% margin)

Responsive Offsets

Apply offsets at specific breakpoints. This column is full width on mobile, but centered on tablet and up:

Full width on mobile
Centered on tablet+
.zCol-12 .zCol-md-6 .zOffset-md-3 
<div class="zRow">
  <div class="zCol-12 zCol-md-6 zOffset-md-3">
    Full on mobile, centered on desktop
  </div>
</div>

Available for all breakpoints: .zOffset-sm-*, .zOffset-md-*, .zOffset-lg-*, .zOffset-xl-*, .zOffset-xxl-*

Remove offset: Use .zOffset-{breakpoint}-0 to reset offset at a specific breakpoint

Common Use Cases

Centered Content

<div class="zRow">
  <div class="zCol-6 zOffset-3">Centered content</div>
</div>

Mobile-First Sidebar

<div class="zRow">
  <!-- Sidebar first on mobile, second on desktop -->
  <div class="zCol-12 zCol-lg-3 zOrder-lg-2">Sidebar</div>
  <div class="zCol-12 zCol-lg-9 zOrder-lg-1">Main</div>
</div>

Spaced Columns

<div class="zRow">
  <div class="zCol-4">Left</div>
  <div class="zCol-4 zOffset-4">Right (with gap)</div>
</div>

Best Practices

  • Semantic HTML - Use ordering for layout only, not semantics
  • Accessibility - Screen readers follow HTML order, not visual order
  • Mobile-first - Default order should make sense on mobile
  • Use offsets - Not empty <div> spacers
  • Math check - Column width + offset should not exceed 12
