Range

Create slider controls for selecting numeric values within a range.

Understanding Range Controls

The <input type="range"> element is a semantic HTML form control that creates a slider for selecting numeric values. It's excellent for accessibility:

Accessibility Benefits:
  • Keyboard navigation: Arrow keys adjust the value, Home/End keys jump to min/max
  • Screen readers: Announce the current value, min/max, and step
  • Touch-friendly: Large draggable area works well on mobile devices
  • Clear constraints: Users can't enter invalid values (always within min/max)

When to use: Volume controls, brightness, price filters, rating scales, zoom levels, or any numeric value within a defined range.

The Range Element

Start with semantic HTML, then enhance with .zRange for consistent, branded styling.

Semantic HTML (Browser Default)

A basic range slider without any zTheme styling - notice the browser's default appearance:

<label for="basicRange">Volume</label>
<input type="range" id="basicRange" min="0" max="100" value="50">

With .zRange Styling

Add .zRange for branded, cross-browser consistent styling:

<label for="styledRange">Volume</label>
<input type="range" class="zRange" id="styledRange" min="0" max="100" value="50">
What .zRange Adds:
  • Custom thumb (draggable circle) with primary color branding (green)
  • Styled track (background line) with rounded corners and shadow
  • Consistent appearance across Chrome, Firefox, and Safari
  • Enhanced focus states for keyboard navigation (green glow)
  • Active state visual feedback when dragging (lighter green)
  • Smooth transitions and animations

Min, Max, and Step

Control the slider's range and precision with min, max, and step attributes.

Custom Range

Set minimum and maximum values (defaults: 0 to 100):

<label for="priceRange">Price Filter: $<span id="priceValue">100</span></label>
<input type="range" class="zRange" id="priceRange" min="20" max="200" value="100">

<script>
  const priceSlider = document.getElementById('priceRange');
  const priceDisplay = document.getElementById('priceValue');
  priceSlider.addEventListener('input', () => {
    priceDisplay.textContent = priceSlider.value;
  });
</script>

Step Increments

Control the increment steps (default: 1). Use step="0.5" for finer control:

<input type="range" class="zRange" min="0" max="5" step="0.5" value="4">

Disabled State

Add the disabled attribute to prevent interaction.

<input type="range" class="zRange" disabled>

Best Practices

Always Use Labels

Associate every range input with a label for accessibility:

<label for="volume" class="zLabel">Volume</label>
<input type="range" class="zRange" id="volume">

Show Current Value

Display the current slider value to users with JavaScript:

<label for="brightness">Brightness: <span id="brightnessValue">50</span>%</label>
<input type="range" class="zRange" id="brightness" value="50">

<script>
  const slider = document.getElementById('brightness');
  const display = document.getElementById('brightnessValue');
  slider.addEventListener('input', () => {
    display.textContent = slider.value;
  });
</script>

Set Meaningful Defaults

Always set a value attribute to show users the initial position:

<!-- Good: Shows starting position -->
<input type="range" class="zRange" value="50">

<!-- Bad: Slider position unclear -->
<input type="range" class="zRange">
Browser Limitation: Firefox supports "filling" the track from the left or right of the thumb. Other browsers don't. zTheme doesn't implement this for cross-browser consistency.
