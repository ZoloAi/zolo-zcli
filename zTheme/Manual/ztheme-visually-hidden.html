Understanding Visually Hidden

The visually hidden technique hides content from sighted users while keeping it accessible to assistive technologies like screen readers. This is different from display: none or visibility: hidden, which hide content from everyone including screen readers.

üí° Why This Matters:

Screen reader users need context that sighted users get visually. For example, a √ó close button is obvious visually, but a screen reader just hears "button" without knowing its purpose. Adding visually hidden text like <span class="zVisually-hidden">Close</span> provides that context!

‚ö†Ô∏è Important Distinction:

  • .zVisually-hidden: Hidden from sight, visible to screen readers
  • display: none: Hidden from everyone (sight AND screen readers)
  • aria-hidden="true": Visible on screen, hidden from screen readers

Best Practices

‚úÖ DO Use For:

  • Icon buttons: Add text for screen readers
  • Skip links: Help keyboard users navigate faster
  • Loading states: Announce "Loading..." to screen readers
  • Form labels: When design shows only placeholders
  • Semantic structure: Headings for screen reader navigation
  • Status updates: Announce form validation messages

‚ùå DON'T Use For:

  • Hiding content from everyone (use display: none)
  • Hiding decorative images (use aria-hidden="true" or empty alt)
  • Critical information that everyone needs (make it visible!)
  • Replacing proper semantic HTML
  • Hiding content from screen readers (use aria-hidden)

üí° Pro Tips:

  • Test with a screen reader (NVDA, JAWS, VoiceOver)
  • Use semantic HTML first, visually hidden second
  • Don't hide critical information from anyone
  • Combine with ARIA attributes when needed
  • Remember: Accessibility benefits everyone!

How It Works

The technique uses multiple CSS properties to hide content visually while keeping it accessible:

Property Value Purpose
position absolute Remove from document flow
width / height 1px Tiny but exists in DOM
padding 0 No internal space
margin -1px Pull completely out of view
overflow hidden Hide any content overflow
clip rect(0,0,0,0) Clip element to nothing
white-space nowrap Prevent text wrapping
border 0 No visible border

üí° Note: This is not display: none or visibility: hidden, which hide content from screen readers too!
