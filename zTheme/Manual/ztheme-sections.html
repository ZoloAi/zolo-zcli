Sections

Styled content sections with color variants, borders, and optional legends. Perfect for organizing content into visually distinct areas.

Understanding Sections

Sections are flexible content containers that provide:

  • Visual Separation: Borders and shadows distinguish content areas
  • Color Coding: Use semantic colors to indicate content type
  • Flexible Layout: Flexbox column layout for easy content stacking
  • Fieldset Styling: Optional legends for labeled sections
Note: For full-viewport hero sections, see Hero Sections. Sections are for organizing content within a page, not for splash screens.

Basic Section

The .zSection class creates a styled content container:

Basic Section

This is a basic section with default styling: light background, border, shadow, and rounded corners.

Content naturally stacks vertically using flexbox column layout.

<div class="zSection">
  <h3>Basic Section</h3>
  <p>This is a basic section with default styling.</p>
</div>

What .zSection provides:

  • 1px border with gray color
  • 8px border radius for rounded corners
  • 1.5rem padding inside
  • 2rem vertical margin between sections
  • Subtle box shadow for depth
  • Flexbox column layout

Getting Started

Sections can combine multiple features for rich, organized content. Use color variants to communicate purpose, legends for labels, and title wrappers for emphasis.

  • Start with the base .zSection
  • Add a color variant like .zSection-info
  • Include a .zLegend for context
  • Use .zTitleWrapper for centered titles

  </div>

</div>

Best Practices

  • Use semantic colors: Match color variants to content purpose (success for confirmations, error for failures)
  • Consistent spacing: Sections have 2rem vertical margins by default - maintain consistency
  • Legend clarity: Keep legend text short and descriptive (1-3 words)
  • Content hierarchy: Use headings (h3, h4) inside sections for structure
  • Don't overuse: Too many sections can fragment your content - use judiciously
  • Color contrast: Ensure text remains readable on colored backgrounds (WCAG AA)
  • Mobile responsive: Sections automatically adjust padding on smaller screens

Usage in zCLI

When using zTheme with zCLI's declarative UI, sections are automatically created for each zKey. You can customize using _zClass metadata:

# Default: Auto-wrapped in .zContainer
zAbout:
  About_Section:
    - zDisplay:
        event: header
        content: "About Us"

# Custom: Use zSection styling instead
zAbout:
  _zClass: "zSection zSection-info"
  About_Section:
    - zDisplay:
        event: header
        content: "About Us"
