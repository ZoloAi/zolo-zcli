Ratio

Maintain perfect aspect ratios for responsive videos, images, and embeds that scale beautifully on any screen size.

Understanding Aspect Ratios

Aspect ratio is the proportional relationship between an element's width and height. A 16:9 ratio means for every 16 units of width, there are 9 units of height. Maintaining aspect ratios ensures media (videos, images) don't get squished or stretched when the screen size changes.

Common Aspect Ratios:

  • 16:9 - Modern video standard (YouTube, Netflix, TV)
  • 4:3 - Classic TV and monitors
  • 1:1 - Square (Instagram posts)
  • 21:9 - Ultrawide cinema
  • 3:2 - Classic photography

üí° How It Works:

The .zRatio container uses a ::before pseudo-element with padding-top to create a percentage-based height. The child element is absolutely positioned to fill this space, maintaining the aspect ratio regardless of width. This is the modern CSS technique for responsive embeds!

Custom Ratios

Create any aspect ratio using the CSS custom property --z-aspect-ratio:

Custom 5:1 Ratio

5:1 Ultra-wide Banner (20%) 
<!-- 5:1 ratio: (1/5)*100% = 20% -->
<div class="zRatio" style="--z-aspect-ratio: 20%;">
  <div>5:1 Ultra-wide Banner</div>
</div>

<!-- 2:1 ratio: (1/2)*100% = 50% -->
<div class="zRatio" style="--z-aspect-ratio: 50%;">
  <div>2:1 Wide Container</div>
</div>

Responsive Aspect Ratios

Change aspect ratios at different breakpoints using CSS custom properties:

/* Start as 4:3, change to 16:9 on large screens */
.zRatio-4x3 {
  --z-aspect-ratio: calc(3 / 4 * 100%); /* 75% */
}

@media (min-width: 992px) {
  .zRatio-4x3 {
    --z-aspect-ratio: calc(9 / 16 * 100%); /* 56.25% */
  }
}

Best Practices

‚úÖ DO:

  • Use 16:9 for modern video content (YouTube, Vimeo)
  • Use 1:1 for square images (Instagram, avatars)
  • Use 3:2 or 4:3 for photo galleries
  • Add title attributes to iframes for accessibility
  • Provide alt text for images inside ratio containers
  • Test responsiveness by resizing your browser
  • Use custom ratios when standard ones don't fit

‚ùå DON'T:

  • Don't use aspect ratios on elements with fixed heights (conflicts!)
  • Don't forget to test with actual content (not just placeholders)
  • Don't use overly extreme ratios (1:10) unless intentional
  • Don't nest ratio containers inside each other (unnecessary)
  • Don't forget allowfullscreen on video iframes
  • Don't use ratios where normal responsive images work fine

Accessibility

Embedded Media

  • Iframes: Always include title attribute describing content
  • Videos: Provide captions/subtitles for accessibility
  • Images: Use descriptive alt text
  • Keyboard: Ensure embedded content is keyboard navigable

Screen Readers

  • Screen readers announce iframe titles and image alt text
  • Ensure video players have accessible controls
  • Provide text alternatives for video-only content
  • Test with screen readers (NVDA, JAWS, VoiceOver)
