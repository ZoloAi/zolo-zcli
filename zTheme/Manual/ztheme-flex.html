Understanding Flexbox

Flexbox is a modern CSS layout system that makes it easy to align, distribute, and arrange elements. Flex utilities work with .zD-flex containers to control alignment, direction, wrapping, and spacing.

ðŸ’¡ Remember: Most flex utilities require a parent with .zD-flex to work! First add .zD-flex, then add flex utilities.

<div class="zD-flex zFlex-center zFlex-items-center">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

Responsive Flex Utilities

Flex utilities support responsive breakpoints (sm, md, lg): 

<!-- Column on mobile, row on tablet+ -->
<div class="zD-flex zFlex-column zFlex-md-row">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

<!-- Center on mobile, space-between on desktop -->
<div class="zD-flex zFlex-center zFlex-lg-between">
  <div>Left</div>
  <div>Right</div>
</div>
Property Responsive Classes
Direction .zFlex-{sm|md|lg}-row, .zFlex-{sm|md|lg}-column
Justify .zFlex-{sm|md|lg}-center, .zFlex-{sm|md|lg}-between
Wrap .zFlex-{sm|md|lg}-wrap, .zFlex-{sm|md|lg}-nowrap
