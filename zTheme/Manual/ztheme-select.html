Select

Style dropdown menus with .zSelect for consistent, accessible selection controls.

The Select Element

The <select> element creates a dropdown menu. zTheme provides minimal reboot styling for semantic HTML, then enhances it with the .zSelect utility class.

Semantic HTML (Reboot Only)

A basic <select> with reboot styling:

<select>
  <option>Select an option</option>
  <option value="1">Option One</option>
</select>

With .zSelect Styling

Add .zSelect for enhanced styling with custom dropdown arrow and consistent appearance:

<select class="zSelect">
  <option>Select an option</option>
  <option value="1">Option One</option>
</select>
What .zSelect Adds: Custom dropdown arrow, consistent padding, border styling, and focus states. The <option> elements cannot be styled due to browser limitations.

Sizing

Match select sizes with text inputs using .zSelect-lg and .zSelect-sm.

<!-- Large -->
<select class="zSelect zSelect-lg">...</select>

<!-- Small -->
<select class="zSelect zSelect-sm">...</select>

Multiple Selection

Allow users to select multiple options with the multiple attribute. This displays all options in a scrollable list.

<select class="zSelect" multiple>
  <option value="1">Engineering</option>
  <option value="2">Design</option>
  <option value="3">Marketing</option>
</select>

Visible Options (Size)

Control how many options are visible at once with the size attribute:

<!-- Show 4 options at once -->
<select class="zSelect" size="4">
  <option value="1">Engineering</option>
  <option value="2">Design</option>
</select>

Disabled State

Add the disabled attribute to prevent user interaction and show a clear visual indication.

<select class="zSelect" disabled>
  <option>Cannot interact with this</option>
</select>
Disabled Styling: Disabled selects use var(--color-error) background with white text to clearly communicate "unavailable" status.

Best Practices

Always Use Labels

Associate every <select> with a <label> for accessibility:

<label for="department" class="zLabel">Department</label>
<select class="zSelect" id="department">...</select>

First Option as Prompt

Use the first option as a prompt, not a selectable value:

<select class="zSelect">
  <option>Select a department</option> ← Prompt
  <option value="eng">Engineering</option> ← Actual choices
</select>

Browser Limitations

Note: The styling applied by .zSelect only affects the <select> element itself. Individual <option> elements cannot be styled due to browser limitations.

CSS Reference

Select elements inherit variables from form controls:

/* Size variants */
.zSelect        /* Default size */
.zSelect-sm     /* Small: 0.875rem */
.zSelect-lg     /* Large: 1.25rem */

/* Key CSS variables */
--form-control-bg: #fff;
--form-control-disabled-bg: var(--color-error);  /* Red for disabled */
--form-control-color: var(--color-darkgray);
--form-control-border-color: var(--color-gray);
