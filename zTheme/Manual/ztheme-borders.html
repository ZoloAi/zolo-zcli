Understanding Border Utilities

Border utilities let you add, remove, color, and size borders on any element. They're perfect for creating visual separation, emphasizing elements, or adding polish to your UI. Border-radius utilities help you round corners for a modern, friendly aesthetic.

üí° Pro Tip: Border utilities can be combined! Use .zBorder with .zBorder-primary and .zBorder-3 to create a 3px primary-colored border.

Best Practices

‚úÖ DO

  • Use .zRounded-circle for avatars and profile images
  • Use .zBorder-top accents to add subtle visual hierarchy
  • Combine border color with matching background for cohesive design
  • Use .zRounded-pill for tags, badges, and modern buttons
  • Round corners on cards, modals, and containers for modern feel

‚ùå DON'T

  • Don't use .zBorder-5 on small elements (looks too heavy)
  • Don't mix different border radii on adjacent elements (inconsistent)
  • Don't forget .zBorder when using color classes alone
  • Don't over-round corners on rectangular content (looks odd)
  • Don't use borders on every element (visual clutter)
