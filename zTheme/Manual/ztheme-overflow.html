Overflow Utilities

Control how content overflows an element's boundaries.

Understanding Overflow

The overflow property controls what happens when content is too large for its container. You can hide overflowing content, add scrollbars, or let it extend beyond the boundaries.

Best Practices

  • Use .zOverflow-auto: Most common choice - scrollbars only when needed
  • Set container dimensions: Overflow requires defined width/height to work
  • Mobile tables: Use .zOverflow-x-auto for wide tables
  • Chat/feeds: Use .zOverflow-y-auto with fixed height
  • Image cropping: Use .zOverflow-hidden for fixed-size containers
  • Dropdowns/tooltips: Use .zOverflow-visible to extend beyond parent
  • Text truncation: Combine .zText-truncate with max-width
  • Long URLs: Use .zText-break to prevent horizontal overflow
  • Performance: Avoid overflow: visible on large scrolling containers

Accessibility

  • Keyboard scrolling: Scrollable areas should be keyboard accessible
  • Focus indicators: Ensure focus is visible in scrollable containers
  • Scroll cues: Provide visual hints that content is scrollable
  • Touch targets: Scrollbars should be wide enough for touch devices
  • Hidden content: Remember .zOverflow-hidden hides content from everyone
  • Screen readers: All content is read regardless of overflow setting
