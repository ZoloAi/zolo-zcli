Icons

Bootstrap Icons - A robust, production-ready icon library for zTheme applications

Understanding the Icon System

zTheme uses Bootstrap Icons - a free, high-quality, open-source icon library with over 2,000 icons. This professional icon system offers:

  • 2,000+ icons - Comprehensive library covering all use cases
  • One-line CDN include - Simple integration, no configuration needed
  • No styling conflicts - Pure icons with no framework dependencies
  • Scalable - Vector-based icons look perfect at any size
  • Customizable - Full control over size, color, and styling via CSS
  • Accessible - Built with accessibility best practices

üí° Why Bootstrap Icons? Maintained by the Bootstrap team, Bootstrap Icons is battle-tested across millions of websites. It's lightweight (~300KB cached across sites), has zero styling conflicts, and provides everything you need without the maintenance burden of a custom icon system.

Basic Usage

Including Icons in Your Page

Add Bootstrap Icons to your project with a single line in your <head>: 

<head>
  <!-- zTheme CSS -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ZoloAi/zTheme@main/dist/ztheme.css">
  
  <!-- Bootstrap Icons -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">
</head>

Using Icons

Icons use simple, intuitive class names with the bi prefix:

Home 

<i class="bi bi-house-fill"></i> Home

Icon Sizes

Control icon size using font-size or zTheme size classes: 

<i class="bi bi-house-fill" style="font-size: 0.875rem;"></i> <!-- Small -->
<i class="bi bi-house-fill" style="font-size: 1rem;"></i>     <!-- Default -->
<i class="bi bi-house-fill" style="font-size: 1.5rem;"></i>   <!-- Large -->
<i class="bi bi-house-fill" style="font-size: 2rem;"></i>     <!-- XL -->
<i class="bi bi-house-fill" style="font-size: 3rem;"></i>     <!-- 2XL -->

Styling Icons

Icon Colors

Icons inherit text color from their parent. Use zTheme color utilities or custom colors: 

<i class="bi bi-house-fill zText-primary"></i>
<i class="bi bi-house-fill zText-success"></i>
<i class="bi bi-house-fill" style="color: #9c27b0;"></i>

<button class="zBtn zBtn-primary">

Finding the Right Icon

Bootstrap Icons provides a searchable gallery at icons.getbootstrap.com

How to Find Icons:

  1. Visit icons.getbootstrap.com
  2. Use the search bar (e.g., "home", "user", "settings")
  3. Click an icon to see its name and variations
  4. Copy the class name (e.g., bi-house-fill)
  5. Use it in your HTML: <i class="bi bi-house-fill"></i>

Icon Naming Convention:

  • bi-icon-name - Outlined version (lighter)
  • bi-icon-name-fill - Filled version (bolder)
  • bi-icon-name-square - Square variant
  • bi-icon-name-circle - Circle variant

Best Practices

‚úÖ Do

  • Use consistent icon styles (all filled OR all outlined, not mixed)
  • Ensure icons have proper contrast against backgrounds (WCAG AA minimum)
  • Add spacing between icons and text (0.25-0.5rem)
  • Use appropriate sizes - not too small (min 16px) or overwhelmingly large
  • Choose meaningful icons that match their purpose
  • Test icons at all responsive breakpoints

‚ùå Don't

  • Mix filled and outlined styles inconsistently
  • Use icons smaller than 14px - they become unclear
  • Rely solely on color to convey meaning (accessibility)
  • Use too many different icons - keep it focused
  • Forget to test icon readability on mobile devices
  • Use decorative icons without proper ARIA labels when needed

Accessibility

Decorative Icons

Icons that are purely decorative (next to text labels) should be hidden from screen readers: 

<button class="zBtn zBtn-primary">
  <i class="bi bi-house-fill" aria-hidden="true"></i>
  Home
</button>

Standalone Icons

If an icon is the ONLY indicator (no text), add an appropriate label: 

<button class="zBtn zBtn-primary" aria-label="Go to home page">
  <i class="bi bi-house-fill"></i>
</button>

Interactive Icons

For clickable icons, ensure proper focus states and touch targets (minimum 44x44px): 

<button 
  class="zBtn" 
  aria-label="Refresh data"
  style="padding: 0.75rem; min-width: 44px; min-height: 44px;">
  <i class="bi bi-arrow-repeat" style="font-size: 1.5rem;"></i>
</button>
