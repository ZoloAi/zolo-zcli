Images

Make images look great on any device with responsive sizing, styling options, and alignment utilities.

The Image Element

The <img> element is semantic HTML for displaying images. Every image should include an alt attribute that describes the image - this is crucial for screen readers and search engines (SEO).

<img src="path/to/image.jpg" alt="Description of the image">

Accessibility: Always include descriptive alt text. Screen readers announce it to visually impaired users, and search engines use it to understand your content.

Mobile-first design: In zTheme, all images are responsive by default - they'll never overflow their container. This is built into the reboot, so you don't need any special classes!

Why Images Just Work

Unlike some frameworks that require you to add classes to every image, zTheme's reboot makes all images responsive automatically. They scale down on smaller screens but never larger than their natural size.

<!-- No special classes needed! -->
<img src="large-photo.jpg" alt="...">

Try it: Resize your browser window - the image above automatically scales down on smaller screens without any utility classes.

Image Thumbnails

Give images a framed appearance with .zImg-thumbnail. This adds a light border, padding, and subtle shadow - perfect for photo galleries or profile pictures.

<img src="..." class="zImg-thumbnail" alt="...">

Multiple Thumbnails

Customizing Thumbnails

Want to change how .zImg-thumbnail looks? These CSS variables in zImages.css control all the styling:

--thumbnail-padding: 0.25rem;
--thumbnail-bg: var(--color-base);
--thumbnail-border-width: 1px;
--thumbnail-border-color: var(--color-border);
--thumbnail-border-radius: 0.25rem;
--thumbnail-box-shadow: 0 0.125rem 0.25rem rgba(0, 0, 0, 0.075);

Override these in your own CSS to customize the thumbnail appearance across your entire site.

Aligning Images

Position images within their containers using alignment utilities. You can float images to wrap text around them, center them, or align them within text. These utilities work on any element - images just benefit from them frequently!

→ For more about Float, Display, and Text Alignment utilities, see Utilities Overview

Text Wrapping

Use .zFloat-start (float left) or .zFloat-end (float right) to make text wrap around images.

Images can be floated to either side to allow text to wrap around them naturally. This is perfect for blog posts, articles, or any content where you want images to integrate smoothly with your text. The float utilities make it easy to position images left or right while maintaining proper spacing and flow. Try resizing your browser to see how the text reflows around the images automatically.

<img src="..." class="zRounded zFloat-start" alt="...">
<img src="..." class="zRounded zFloat-end" alt="...">

Centering Images 1

Center an image by making it a block element (.zD-block) and setting auto horizontal margins (.zMx-auto).

<img src="..." class="zRounded zMx-auto zD-block" alt="...">

Centering Images 2

Alternatively, add .zText-center to the parent container. This centers all inline content inside, including images.

<div class="zText-center">
  <img src="..." class="zRounded" alt="...">
</div>

Picture Element

The <picture> element is semantic HTML that displays different images based on conditions. It's perfect for two scenarios: showing different image crops for different screen sizes, or providing modern image formats with fallbacks for older browsers.

Responsive Image Crops

Sometimes you want different image crops for different screen sizes - like a wide landscape photo on desktop but a taller portrait crop on mobile. The <picture> element makes this easy with media queries.

<picture>
  <source media="(min-width: 768px)" srcset="landscape.jpg">
  <source media="(max-width: 767px)" srcset="portrait.jpg">
  <img src="landscape.jpg" alt="...">
</picture>

How it works: The browser checks each <source> from top to bottom. When it finds a matching media query, it uses that image. If no media query matches (or the browser doesn't support <picture>), it falls back to the <img> tag.

In this example: screens 768px or wider get the wide landscape image, smaller screens get the portrait crop, and older browsers always get the landscape image as a fallback.

Try it: Resize your browser window to see the image switch between landscape and portrait crops! Notice it's the same photo, just cropped differently.

Modern Format Fallbacks

WebP images are typically 25-35% smaller than JPG with the same quality, but older browsers don't support them. Use <picture> to serve WebP to modern browsers and JPG to older ones.

<picture>
  <source srcset="photo.webp" type="image/webp">
  <img src="photo.jpg" alt="...">
</picture>

How it works: The browser checks if it can display WebP (by checking the type attribute). If yes, it uses the WebP image. If no, it falls back to the JPG in the <img> tag. Everyone sees a picture, but modern browsers get smaller file sizes!

Important: Always apply utility classes (like .zImg-thumbnail or .zRounded) to the <img> element inside <picture>, not to the <picture> element itself.

Object Fit

When an image and its container have different aspect ratios, use object-fit utilities to control how the image fills the space. Think of it like choosing how a photo fits in a frame.

.zObject-fit-contain

.zObject-fit-cover

.zObject-fit-fill

.zObject-fit-scale

<img src="..." class="zObject-fit-contain" alt="...">
<img src="..." class="zObject-fit-cover" alt="...">
<img src="..." class="zObject-fit-fill" alt="...">
<img src="..." class="zObject-fit-scale" alt="...">

Which one to use?

  • .zObject-fit-contain - Shows entire image (may have empty space)
  • .zObject-fit-cover - Fills container completely (may crop image) ⭐ Most common
  • .zObject-fit-fill - Stretches image to fill (may distort)
  • .zObject-fit-scale - Like contain, but won't upscale small images

Common Use Cases

Here are some real-world examples combining the techniques you've learned. Copy and adapt these for your own projects!

Profile Picture

Use .zRounded-circle for circular profile images. Combine with flexbox utilities for nice user cards.

John Doe

Software Engineer

<div style="display: flex; align-items: center; gap: 1rem;">
  <img src="profile.jpg" class="zRounded-circle" alt="Profile picture">
  <div>
    <h4>John Doe</h4>
    <p>Software Engineer</p>
  </div>
</div>

Photo Gallery

Use .zImg-thumbnail with CSS Grid for responsive galleries that look great on any screen.

<div style="display: grid; grid-template-columns: repeat(auto-fill, minmax(150px, 1fr)); gap: 1rem; justify-items: center;">
  <img src="photo1.jpg" class="zImg-thumbnail" alt="...">
  <img src="photo2.jpg" class="zImg-thumbnail" alt="...">
  <img src="photo3.jpg" class="zImg-thumbnail" alt="...">
</div>

Hero Banner

Use .zObject-fit-cover to fill a fixed-height container with an image, perfect for hero sections.

Welcome to zTheme

Beautiful, responsive images

<div style="position: relative; height: 300px;">
  <img src="hero.jpg" class="zObject-fit-cover" alt="..." 
       style="width: 100%; height: 100%;">
  <div style="position: absolute; top: 50%; left: 50%; ...">
    <h2>Your Heading</h2>
  </div>
</div>

Tip: Choose hero images with natural depth (forests, mountains, skies) for beautiful backgrounds. White text with a strong dark shadow works best for readability across various image themes.
