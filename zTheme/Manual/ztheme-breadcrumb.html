Breadcrumb

Indicate the current page's location within a navigational hierarchy with automatic CSS separators.

Understanding Breadcrumbs

Breadcrumbs show users where they are in your site's structure, making it easy to navigate back to parent pages. They're perfect for:

  • Deep navigation: Helping users understand their location in multi-level structures
  • Quick backtracking: Easy navigation to parent pages without browser back button
  • SEO context: Search engines use breadcrumbs to understand site structure
  • User orientation: Reducing confusion in complex applications or documentation
Accessibility: Breadcrumbs provide navigation context. Wrap them in a <nav> element with aria-label="breadcrumb" to describe the navigation type, and add aria-current="page" to the current page item.

The Semantic Foundation

Breadcrumbs are built using semantic HTML: a <nav> element containing an ordered list (<ol>). Without styling, this creates a simple numbered list:

  1. Home
  2. Library
  3. Data
<nav aria-label="breadcrumb">
  <ol>
    <li><a href="#">Home</a></li>
    <li><a href="#">Library</a></li>
    <li>Data</li>
  </ol>
</nav>
Why <ol>? Ordered lists semantically indicate a sequence—perfect for breadcrumbs showing "step 1, step 2, step 3" navigation. Screen readers announce "list with 3 items," helping users understand the navigation depth.

Enhanced Breadcrumb with .zBreadcrumb

The .zBreadcrumb class transforms the basic list into a horizontal navigation trail with automatic separators. Compare this to the unstyled example above:

  1. Home
  2. Library
  3. Data
<nav aria-label="breadcrumb">
  <ol class="zBreadcrumb">
    <li class="zBreadcrumb-item"><a href="#">Home</a></li>
    <li class="zBreadcrumb-item"><a href="#">Library</a></li>
    <li class="zBreadcrumb-item zActive" aria-current="page">Data</li>
  </ol>
</nav>

The transformation: .zBreadcrumb removes list styling, arranges items horizontally, adds separators between items, and styles the active (current) page.

zTheme Pattern: The <nav> and <ol> provide semantic structure (SEO and accessibility). The .zBreadcrumb utility classes add professional visual design. Separation of concerns!

Custom Dividers

Change the separator between breadcrumb items using CSS variables or utility classes. The default is /, but you can customize it:

Arrow Divider

  1. Home
  2. Library
<nav aria-label="breadcrumb" style="--breadcrumb-divider: '>';">
  <ol class="zBreadcrumb">
    <li class="zBreadcrumb-item"><a href="#">Home</a></li>
    <li class="zBreadcrumb-item zActive" aria-current="page">Library</li>
  </ol>
</nav>

Bullet Divider

  1. Home
  2. Products

No Divider

  1. Home
  2. Settings
<!-- No divider: set to empty string -->
<nav aria-label="breadcrumb" style="--breadcrumb-divider: '';">
  <ol class="zBreadcrumb">
    <li class="zBreadcrumb-item"><a href="#">Home</a></li>
    <li class="zBreadcrumb-item zActive">Settings</li>
  </ol>
</nav>

Tip: You can use any character or even emoji as dividers: →, », ⟩, |

Best Practices

  • Use semantic HTML: Always wrap breadcrumbs in <nav aria-label="breadcrumb">
  • Mark current page: Add aria-current="page" to the last item and .zActive class
  • Don't link the current page: The last breadcrumb item should be plain text, not a link
  • Start with Home: The first breadcrumb typically links to the homepage or dashboard
  • Keep labels short: Use concise page names (e.g., "Docs" instead of "Documentation Center")
  • Show parent hierarchy: Include all parent levels, not just the immediate parent
  • Mobile-friendly: Consider truncating middle items on small screens if the path is very long
  • Consistent placement: Place breadcrumbs at the top of the page, before the main heading

CSS Reference

Breadcrumb Classes

Class Description
.zBreadcrumb Base container for breadcrumb navigation (apply to <ol>)
.zBreadcrumb-item Individual breadcrumb item (apply to each <li>)
.zActive Marks the current page (muted color, no underline)

CSS Variables

Variable Default Description
--breadcrumb-divider "/" Character or symbol between breadcrumb items
