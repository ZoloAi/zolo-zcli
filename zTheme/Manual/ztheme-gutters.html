Gutters

Control spacing between columns

What Are Gutters?

Gutters are the spacing between columns in a grid. They create visual separation without requiring margin utilities.

  • Default - 0.75rem (12px) horizontal padding
  • Responsive - Can change at different breakpoints
  • Directional - Control horizontal, vertical, or both

Common Use Cases

Card Grids with No Gaps

<div class="zRow zG-0">
  <div class="zCol-4">Card</div>
  <div class="zCol-4">Card</div>
  <div class="zCol-4">Card</div>
</div>

Tight Mobile, Spacious Desktop

<div class="zRow zG-1 zG-lg-4">
  <div class="zCol-6 zCol-lg-3">...</div>
</div>

Custom Vertical Spacing

<div class="zRow zGy-5">
  <!-- Large vertical gaps between wrapped rows -->
</div>

Gutter Classes Reference

Type Classes Effect
All .zG-0 to .zG-5 Horizontal spacing (default)
Horizontal .zGx-0 to .zGx-5 Left/right only
Vertical .zGy-0 to .zGy-5 Top/bottom only
Responsive .zG-{breakpoint}-* sm, md, lg, xl, xxl

Best Practices

  • Default is good - Use .zG-3 (0.75rem) for most layouts
  • No gaps for images - Use .zG-0 for seamless photo galleries
  • Vertical spacing - Add .zGy-* for wrapped/multi-row grids
  • Responsive spacing - Tighter on mobile, more generous on desktop
  • Consistency - Use same gutter sizes throughout your design system
