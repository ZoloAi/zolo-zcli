Vertical Alignment

Control vertical alignment of inline and table elements

Understanding Vertical Alignment

Vertical alignment utilities control how inline, inline-block, inline-table, and table cell elements align vertically within their container.

‚ö†Ô∏è Important: These utilities only work on inline, inline-block, inline-table, and table cell elements. For vertically centering block elements like <div>, use Flexbox utilities instead!

Inline Elements

Align inline elements like <span> and <img> within their line:

baseline top middle bottom text-top text-bottom 
<span class="zAlign-baseline">baseline</span>
<span class="zAlign-top">top</span>
<span class="zAlign-middle">middle</span>
<span class="zAlign-bottom">bottom</span>
<span class="zAlign-text-top">text-top</span>
<span class="zAlign-text-bottom">text-bottom</span>

‚ö†Ô∏è Note: text-top and text-bottom align to the font's ascender/descender lines, not the line box. This causes counterintuitive positioning (text-top appears below where you'd expect). Use .zAlign-top, .zAlign-middle, and .zAlign-bottom for predictable results.

Table Cells

Control vertical alignment within table cells (especially useful for tall rows):

Baseline Top Middle Bottom
baseline top middle bottom 
<table class="zTable">
  <tr style="height: 150px;">
    <td class="zAlign-baseline">baseline</td>
    <td class="zAlign-top">top</td>
    <td class="zAlign-middle">middle</td>
    <td class="zAlign-bottom">bottom</td>
  </tr>
</table>

Note: .zAlign-text-top and .zAlign-text-bottom also exist but are not recommended for table cells due to unpredictable font-metric behavior.

Best Practices

‚úÖ Do

  • Use .zAlign-middle for icons next to text
  • Use .zAlign-top for table cells with varying content heights
  • Use .zAlign-middle for centering small images inline
  • Combine with line-height for fine-tuned control

‚ùå Don't

  • Try to use on block elements (<div>, <p>) - won't work!
  • Use for centering page layouts - use Flexbox instead
  • Forget that alignment is relative to the line or cell height
  • Overcomplicate - .zAlign-middle solves 90% of use cases

üí° Pro Tip: For centering block-level content vertically, use .zFlex-column .zJustify-center instead. Vertical alignment utilities are specifically for inline and table contexts.

Accessibility

  • Visual only: Alignment is purely visual and doesn't affect screen readers
  • Icons: When aligning icons, ensure they have proper aria-label or are decorative
  • Images: Always include alt text on aligned images
  • Tables: Use proper <th> and scope attributes regardless of alignment
