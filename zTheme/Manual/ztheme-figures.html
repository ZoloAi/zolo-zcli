Figures

Use semantic HTML elements to display images with captions

The Figure Element

The <figure> element is semantic HTML for displaying images (or other content) with an optional caption. It tells screen readers and search engines that the image and its caption are related.

When to use figures:

  • Images in articles or blog posts that need context
  • Diagrams, charts, or illustrations with explanatory text
  • Photos that need photographer credits or descriptions
  • Portfolio work with project titles and details

Basic Semantic Figure

Here's a figure using just semantic HTML (no styling classes). The reboot provides minimal default styling - bottom margin and muted caption color:

<figure>
  <img src="..." alt="A scenic forest landscape">
  <figcaption>A scenic forest landscape captured at dawn.</figcaption>
</figure>

Semantic importance: The <figure> element creates a relationship between the image and caption. Screen readers announce this connection, helping users understand that the caption describes the image.

Enhanced Styling with zFigure

zTheme provides utility classes for enhanced figure styling. Use .zFigure for inline-block display and .zFigure-caption for consistent caption styling:

<figure class="zFigure">
  <img src="..." class="zFigure-img zImg-fluid zRounded" alt="...">
  <figcaption class="zFigure-caption">A scenic forest landscape captured at dawn.</figcaption>
</figure>

What the utility classes add:

  • .zFigure - Sets display to inline-block (figure only takes up space it needs)
  • .zFigure-img - Adds margin below image to separate it from caption
  • .zFigure-caption - Applies consistent font size and color
  • .zImg-fluid - Makes image responsive (from Images page)
  • .zRounded - Adds rounded corners (from Images page)

Caption Alignment

Captions are left-aligned by default. Use text alignment utilities to change caption position:

<!-- Left (default) -->
<figcaption class="zFigure-caption">Left aligned (default)</figcaption>

<!-- Center -->
<figcaption class="zFigure-caption zText-center">Center aligned</figcaption>

<!-- Right -->
<figcaption class="zFigure-caption zText-end">Right aligned</figcaption>

Image Styling

Combine figures with image utilities from the Images page for different visual effects:

<!-- Rounded corners -->
<img class="zFigure-img zImg-fluid zRounded" alt="...">

<!-- Thumbnail border -->
<img class="zFigure-img zImg-fluid zImg-thumbnail" alt="...">

Figure Galleries

Display multiple figures together using CSS Grid. The grid automatically adjusts columns based on screen width:

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 1.5rem; justify-items: center;">
  <figure class="zFigure">
    <img src="..." class="zFigure-img zImg-fluid zRounded" alt="...">
    <figcaption class="zFigure-caption zText-center">Mountain peak at sunset</figcaption>
  </figure>
  <figure class="zFigure">
    <img src="..." class="zFigure-img zImg-fluid zRounded" alt="...">
    <figcaption class="zFigure-caption zText-center">Ocean waves crashing</figcaption>
  </figure>
  <figure class="zFigure">
    <img src="..." class="zFigure-img zImg-fluid zRounded" alt="...">
    <figcaption class="zFigure-caption zText-center">Desert sand dunes</figcaption>
  </figure>
</div>

Accessibility

Alt text vs. caption: They serve different purposes!

  • Alt text - Describes what's IN the image (for blind users)
  • Caption - Adds context, credits, or explanatory information (for everyone)

Example: Alt text: "Hiker on mountain trail at dawn" | Caption: "Photo credit: Outdoor Adventures Magazine"

Why use semantic <figure>? Screen readers announce the relationship between the image and caption, making your content more accessible.

CSS Reference

Figure styles are defined in zFigures.css and zReboot.css:

Reboot (Semantic HTML)

figure {
  margin: 0 0 1rem;
}

figcaption {
  font-size: 0.875em;
  color: var(--color-gray);
}

Utility Classes

  • .zFigure - Sets display to inline-block
  • .zFigure-img - Adds 0.5rem margin below image
  • .zFigure-caption - Consistent font size and color
  • .zText-start, .zText-center, .zText-end - Caption alignment

CSS Variables

--figure-caption-font-size: 0.875em;
--figure-caption-color: var(--color-gray);
