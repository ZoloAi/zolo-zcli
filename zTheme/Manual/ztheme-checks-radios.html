Checks Radios

Create accessible checkboxes, radio buttons, and toggle switches for selecting options.

Understanding Form Controls

Checkboxes and radio buttons are semantic HTML elements with built-in meaning and behavior. They're not just visual styling—they're essential for:

  • Accessibility: Screen readers announce "checkbox, checked" or "radio button, 2 of 3" to help users understand the control type and state
  • Keyboard Navigation: Users can navigate with Tab and select with Space/Enter without a mouse
  • Form Submission: Browsers automatically include checked values when forms are submitted
  • Native Behavior: Radio buttons automatically uncheck others in the same group—you don't write JavaScript for this
Important: The type attribute (type="checkbox" or type="radio") defines the semantic meaning. The visual styling is secondary. zTheme enhances the appearance while preserving all native functionality.

Semantic HTML (Reboot Only)

Basic checkboxes and radios with minimal reboot styling:

<!-- Semantic checkbox -->
<input type="checkbox" id="basicCheck1" value="terms">
<label for="basicCheck1">I agree to the terms</label>

<!-- Semantic radio buttons (same name = group) -->
<input type="radio" name="basicRadio" id="basicRadio1" value="yes">
<label for="basicRadio1">Yes</label>
<input type="radio" name="basicRadio" id="basicRadio2" value="no">
<label for="basicRadio2">No</label>

This works perfectly for forms, accessibility, and SEO. Now let's enhance the appearance with zTheme classes.

Styled Checkboxes

Add .zCheck wrapper with .zCheck-input and .zCheck-label classes for enhanced, custom-styled checkboxes.

With .zCheck Styling

<div class="zCheck">
  <input class="zCheck-input" type="checkbox" id="checkbox1">
  <label class="zCheck-label" for="checkbox1">
    I agree to the terms and conditions
  </label>
</div>
What .zCheck Adds: Custom checkmark icon, consistent sizing, better spacing, focus states, and visual feedback. The semantic meaning and native behavior remain unchanged.
Accessibility: Always pair <input> with <label> using matching id and for attributes. This creates a larger click area and announces the label text to screen readers.

Disabled State

Add the disabled attribute to prevent interaction.

<input class="zCheck-input" type="checkbox" disabled>

Styled Radio Buttons

Radio buttons allow users to select one option from a group. All radios in a group must share the same name attribute. Add .zCheck wrapper for enhanced styling.

<!-- All radios share name="paymentMethod" -->
<div class="zCheck">
  <input class="zCheck-input" type="radio" name="paymentMethod" id="radio1" checked>
  <label class="zCheck-label" for="radio1">Credit Card</label>
</div>
<div class="zCheck">
  <input class="zCheck-input" type="radio" name="paymentMethod" id="radio2">
  <label class="zCheck-label" for="radio2">PayPal</label>
</div>

Disabled State

<input class="zCheck-input" type="radio" disabled>

Toggle Switches

Switches are styled checkboxes that look like toggle switches. Add .zSwitch alongside .zCheck for the toggle appearance.

<div class="zCheck zSwitch">
  <input class="zCheck-input" type="checkbox" id="switch1">
  <label class="zCheck-label" for="switch1">Enable notifications</label>
</div>

Inline Layout

Display checkboxes or radios horizontally by adding .zCheck-inline to the .zCheck wrapper.

Inline Checkboxes

<div class="zCheck zCheck-inline">
  <input class="zCheck-input" type="checkbox" id="inline1">
  <label class="zCheck-label" for="inline1">Red</label>
</div>
<div class="zCheck zCheck-inline">
  <input class="zCheck-input" type="checkbox" id="inline2">
  <label class="zCheck-label" for="inline2">Green</label>
</div>

Inline Radio Buttons

<div class="zCheck zCheck-inline">
  <input class="zCheck-input" type="radio" name="priority" id="priorityLow">
  <label class="zCheck-label" for="priorityLow">Low</label>
</div>

Best Practices

Always Use Labels

Connect inputs with labels for accessibility and larger click areas:

<input class="zCheck-input" id="myCheckbox">
<label class="zCheck-label" for="myCheckbox">Label text</label>

Radio Button Grouping

All radio buttons in a group must share the same name attribute to work correctly.

Choose the Right Control

  • Checkbox: Select multiple options (0, 1, or more)
  • Radio: Select one option from a group (exactly 1)
  • Switch: Enable/disable a feature (on/off)
