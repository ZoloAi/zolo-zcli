Utilities

Single-purpose classes for rapid UI development

What Are Utilities?

Utilities are single-purpose CSS classes that apply one specific style. They enable rapid prototyping and consistent styling without writing custom CSS.

Example

<div class="zm-3 zp-4 text-center">
  Margin: 1rem, Padding: 1.5rem, Centered text
</div>

Naming Convention

All utility classes follow a consistent pattern:

.z{Property}-{Value}

Examples

  • .zm-3 → Margin: 1rem
  • .zW-50 → Width: 50%
  • .zPosition-relative → Position: relative
  • .zFlex-center → Justify-content: center

Directional Modifiers

For spacing utilities (margin/padding):

  • t = top
  • r = right
  • b = bottom
  • l = left
  • x = horizontal (left + right)
  • y = vertical (top + bottom)

Spacing Scale

All spacing utilities use a six-level scale based on 1rem:

Level Value Pixels Example
0 0 0px .zm-0
1 0.25rem 4px .zm-1
2 0.5rem 8px .zm-2
3 1rem 16px .zm-3
4 1.5rem 24px .zm-4
5 3rem 48px .zm-5

Quick Examples

Centering Content

<div class="zD-flex zFlex-center zFlex-items-center" style="height: 200px;">
  Centered both ways
</div>

Spacing Around Card

<div class="zm-4 zp-3">
  Card with margin outside, padding inside
</div>

Full Width, Half Height

<div class="zW-100 zVh-50">
  Hero section
</div>

Sticky Header

<header class="zPosition-sticky zTop-0 zZ-fixed">
  Always on top
</header>

Best Practices

  • Consistency - Use utilities for common patterns
  • Composition - Combine utilities instead of custom CSS
  • Readability - Don't overload with too many classes
  • Components - Extract repeated patterns
  • !important - Utilities use !important for reliability
