About zBlog

zBlog is a demonstration project that mirrors Corey Schafer's Flask Blog Tutorial, but built with zCLI instead of Flask.

Why zCLI?

zCLI is a declarative Python framework that reduces boilerplate by 90% compared to traditional web frameworks like Flask or Django.

Key Features

  • Declarative Routes: Define routes in YAML instead of Python decorators
  • Jinja2 Templates: Same templating engine as Flask
  • WebSocket Built-in: Real-time communication without Flask-SocketIO
  • Auto-Rendering: BifrostClient auto-renders zDisplay events
  • Schema-Driven: Define database models in YAML, not Python classes
